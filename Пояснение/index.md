```jsx
jsximport { StrictMode } from 'react'                              // Режим строгой проверки (помогает находить баги на этапе разработки)
import { createRoot } from 'react-dom/client'                     // Новый способ рендера React 18
import './index.css'                                              // Глобальные стили для всего проекта
import App from './App.jsx'                                       // Главный компонент — всё приложение

createRoot(document.getElementById('root')).render(           // Находим <div id="root"> в index.html и монтируем React туда
  <StrictMode>                                                // Включаем строгий режим только в разработке (в продакшене он
                                                                  отключается автоматически)  
    <App />                                                   // Запускаем наше приложение
  </StrictMode>,
)
`
Это самый первый файл, который запускается при открытии сайта
createRoot — современный способ (с React 18), поддерживает Concurrent Mode и автоматический батчинг
<StrictMode> — не влияет на продакшен, но в разработке:
Дважды вызывает некоторые эффекты и рендеры (чтобы выявить баги)
Предупреждает о небезопасных практиках

После выполнения этого файла управление передаётся в App.jsx
